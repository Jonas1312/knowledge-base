# Bahasa Indonesian

I:

- saya: I (polite)
- aku: I (informal)

Nama saya Jonas: My name is Jonas

- nama: name
- saya: I (polite)

Saya orang Prancis: I am French

- orang: person
- Prancis: France

Saya tidak suka durian: I don't like durian

- tidak: no, not
- suka: like

Sampai jumpa: See you later

- sampai: until
- jumpa: meet
